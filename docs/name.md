# The name "Edo"

The name works on two levels:

**1. e(lectronic) + do**
It's a contraction of "electronic do" — as in, an agent that *does* things electronically. Written and said as "edo", not "e-do".

**2. A Malayalam interjection**
In Malayalam (the native language of Kerala, India), *എടോ* ("edo") is a casual attention-grabber — roughly equivalent to "hey" or "yo". A friend would say *"edo, can we do that?"* to get your attention before asking something. The app borrows that same energy: a familiar, conversational way to get your agent's attention.

The two meanings reinforce each other — it's both a functional description and a natural way to address something you're about to ask a favour of.
